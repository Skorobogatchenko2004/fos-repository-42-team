# Данные (вход к КИМ-4.1)

Три хронологических среза продаж розничной сети. Разбиение выполнено по времени.

| Файл | Назначение | Период |
|---|---|---|
| `sales_train.csv` | обучающая выборка (использована командой разработки) | ранний |
| `sales_test.csv` | отложенная выборка (использована для отчёта команды) | следует за обучающей |
| `sales_fresh.csv` | свежий срез для проверки дрейфа | последний |

## Колонки

| Колонка | Описание |
|---|---|
| `day` | номер дня (сквозная нумерация) |
| `store_id` | идентификатор магазина |
| `store_format` | формат магазина (hyper, super, conv, disc) |
| `category` | товарная категория (cat_A … cat_H) |
| `dow` | день недели (0–6) |
| `month` | месяц (0–11) |
| `promo_flag` | признак промоакции (0/1) |
| `lag_7`, `lag_14` | продажи 7 и 14 дней назад |
| `roll_28_mean` | скользящее среднее продаж за 28 дней |
| `sales` | целевая переменная — продажи за день |

Признаки `lag_7`, `lag_14`, `roll_28_mean` — те же, что использованы моделью. Целевая переменная — `sales`.
