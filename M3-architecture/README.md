# Модуль 3. Архитектура систем

Модуль формирует навыки проектирования архитектуры систем искусственного интеллекта: применение архитектурных паттернов, trade-off анализ и выбор технологического стека, модельно-ориентированная системная инженерия (MBSE) и разработка архитектурного описания с обеспечением трассируемости к требованиям.

**Проверяемые индикаторы:** LC-3.1 (С), LC-1.2 (С), ОПК-4.1 (Б).
