# Модуль 2. Управление требованиями

Модуль формирует навыки сбора, анализа, документирования требований к системам искусственного интеллекта и управления их изменениями. Особое внимание уделяется специфике требований для ИИ-систем — требованиям к данным, качеству модели, дрейфу и объяснимости, а также обеспечению сквозной трассируемости.

**Проверяемые индикаторы:** LC-1.1 (С), LC-1.3 (С), LC-1.2 (Б).
