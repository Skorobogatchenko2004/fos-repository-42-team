# Датасеты

Учебные и проектные данные дисциплины. Для каждого датасета указываются источник, лицензия, состав признаков, целевая переменная, объём, ограничения, возможные смещения и рекомендуемое разбиение.

| Название | Аннотация | Связанные КИМ | Доступ | Лицензия | Дата проверки |
|---|---|---|---|---|---|
| Продажи розничной сети (прогнозирование спроса) | Исторические данные продаж для сквозного кейса дисциплины; используются для построения контура проверки модели | КИМ-4.1, Проект | открытый датасет прогнозирования спроса либо данные партнёра | по источнику | 17.07.2026 |

## Требования к применению

Для КИМ-4.1 датасет сопровождается тремя срезами: обучающая выборка, тестовая выборка и хронологически отложенный свежий срез для проверки дрейфа. Разбиение выполняется по времени, а не случайно.

Датасет с намеренно внесённым дефектом (утечка признака, смещение, деградация во времени) готовится преподавателем скриптом на основе открытого набора; описание дефекта относится к [материалам преподавателя](../../methodical-guidelines/teachers-assessment/README.md).
