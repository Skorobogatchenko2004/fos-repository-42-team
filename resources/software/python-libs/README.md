## Требования к применению

Лабораторные работы выполняются на фиксированных версиях (lockfile) с отключённой телеметрией. Библиотеки Evidently (смена API в версии 0.7), Deepchecks и NannyML (конфликты зависимостей) требуют внимания при установке; Alibi Detect (лицензия BSL) в учебном применении не используется. Выбор библиотеки обучающимся обосновывается по критерию LC-1.2.
