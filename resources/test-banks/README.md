# Банки тестовых заданий

В текущей версии дисциплины закрытые тестовые задания не применяются. Текущий контроль проводится в форме защиты лабораторных работ, практических заданий и деловых игр; промежуточная аттестация — зачёт с защитой проектной работы. Выбор в пользу проверки понимания на защите обусловлен спецификой дисциплины: проверяемые артефакты являются документами, для которых тестовый контроль не подтверждает освоение.

При включении тестовых заданий в последующие версии для каждого банка указываются темы, количество заданий, уровни сложности, правила выборки, ключи и порядок обновления.
