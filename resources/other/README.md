# Стандарты и нормативная база

Базовая совокупность стандартов, используемая при изучении дисциплины и проверяемая по индикатору ОПК-4.1. Номера и редакции проверены 17.07.2026 и подлежат ежегодной сверке перед началом семестра. Ссылка на устаревшую редакцию стандарта в работе обучающегося является дефектом оформления.
