# Модуль 1. Введение в системную инженерию

Модуль формирует базовые понятия системной инженерии, знакомит с отличиями систем искусственного интеллекта от традиционного программного обеспечения, моделями жизненного цикла и методами оценки целесообразности применения искусственного интеллекта.

**Проверяемые индикаторы:** ОПК-4.1 (Б), LC-4.1 (Б), LC-1.1 (Б).
